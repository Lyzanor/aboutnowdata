About / Data / Now

An agentic system, for humans.

About holds who you are — values, rules, identity. Rarely changes. Data holds what you know — projects, references, structured detail. Now holds what matters today — priorities, context, next steps.

Any AI agent can read this before it acts. What it reads is exactly what you wrote. No inference. No drift. No surprises.

Why not automatic memory

Most memory systems infer and consolidate silently. You don't see what they store. When they're wrong, you find out late.

This is the opposite. You write it. You own it. Plain text, any editor, any agent.

Don't make humans learn the system. Make the system understand the human.
