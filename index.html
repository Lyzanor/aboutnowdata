Personal memory framework

About Now Data

An agentic system, for humans.

About holds who you are. Now holds what matters today. Data holds what you know.

Any agent reads exactly what you wrote before it acts. No inference. No drift. No surprises.

Why not automatic memory

Most memory systems infer and consolidate silently. You don't see what they store.

This is the opposite. You write it. You own it.

Don't make humans learn the system. Make the system understand the human.

Stable layer

About.md is the durable core.

Click the card and edit directly.

Identity, values, boundaries, and style stay readable, editable, and versionable. This is the file an agent should trust before it decides how to behave.

What it is

A human-readable interface between memory, context, and action.

About Now Data is a lightweight structure for keeping stable identity, durable knowledge, and current priorities separate enough for humans to maintain and simple enough for any agent to read directly.
